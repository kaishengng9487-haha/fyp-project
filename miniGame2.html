How to Play

Click on the game area to drop Korean characters (jamos). When jamos touch, they merge to form syllables and words.

Complete all target words to win the game!

Merge Rules:

  • ㄴ + ㅏ = 나
  • ㅅ + ㅏ = 사
  • ㅈ + ㅏ = 자
  • ㅅ + ㅗ = 소
  • ㅁ + ㅜ = 무
  • 무 + ㄹ = 물
  • ㅊ + ㅏ = 차
  • 소 + ㄴ = 손

Try to make:

  • 나무 (Tree)
  • 물 (Water)
  • 사자 (Lion)
  • 차 (Car)
  • 손 (Hand)
Click to drop jamos. Merge them to form syllables and words!

Game Description

About This Game

This educational hangeul merging game combines the addictive gameplay of Suika (Watermelon) Game with Korean language learning. Start with basic jamo characters and merge them to form syllables, then combine syllables to create complete Korean words. It's a fun and interactive way to understand how hangeul characters combine to form words!

How to Play

Click on the game area to drop Korean characters (jamos). When jamos touch, they merge to form syllables and words.

Complete all target words to win the game!

Merge Rules (examples):

  • ㄴ + ㅏ = 나
  • ㄹ + ㅏ = 라
  • ㄹ + ㅣ = 리
  • ㅁ + ㅓ = 머
  • ㅎ + ㅏ = 하
  • ㅁ + ㅏ = 마
  • ㅅ + ㅗ = 소
  • ㅅ + ㅜ = 수
  • ㅁ + ㅜ = 무

Try to make:

  • 나라 (nation)
  • 머리 (head)
  • 하마 (hippo)
  • 소나무 (pine tree)
  • 수리 (repair)
  • 소리 (sound)
Click to drop jamos. Merge them to form syllables and words!

Game Description

About This Game

This educational hangeul merging game combines Suika-like gameplay with Korean language learning. Merge jamos to make syllables, merge syllables to make words!

Commonly-used jamos and syllables will always be available for forming words.
